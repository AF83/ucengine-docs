# Whiteboard

## Options

* ucemeeting: current meeting with attached session
* disabled: readonly mode
* width
* height
* canvas_id_bottom
* canvas_id_interface
* controls_id
* choosers_id
* widget_color: show/hide color widget
* widget_color_id
* widget_linewidth: show/hide linewidth widget
* widget_linewidth_id
* widget_transport: show/hide transport widget
* widget_transport_id

## Methods

* hideControls
* showControls
* clear
* destroy

## Examples

```javascript
$("#demo").whiteboard();
```
